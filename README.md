# BizLedger

A clean, minimal finance tracker built for freelancers and small business owners. Track income, expenses, taxes, and budget allocations — all in one place.

Built with **React Native** and **Expo**.

---

## Features

- **Income & Expense Tracking** — Log transactions with categories, descriptions, and dates
- **Tax Estimation** — Real-time federal, self-employment, and state tax estimates based on your actual profit
- **Budget Allocations** — See your after-tax income split across four budgeting frameworks (50/30/20, 70/20/10, Dave Ramsey, and FIRE)
- **Recurring Transactions** — Set up weekly, monthly, or yearly recurring entries with due date reminders
- **CSV Export** — Export your full transaction history to a spreadsheet in one tap
- **Search & Filter** — Find transactions by description, category, type, or date range
- **Persistent Storage** — Your data stays on your device across sessions
- **Onboarding** — Clean first-launch walkthrough for new users

## Getting Started

### Prerequisites
- Node.js v20+
- Expo Go app on your phone ([iOS](https://apps.apple.com/app/expo-go/id982107779) / [Android](https://play.google.com/store/apps/details?id=host.exp.exponent))

### Installation

```bash
git clone https://github.com/yourusername/BizLedger.git
cd BizLedger/ledger
npm install
npx expo start
```

Scan the QR code with Expo Go and the app will load on your device.
